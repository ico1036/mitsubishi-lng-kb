# F · 밸류체인 단계별 한·일 비교

**클레임:** 차이는 '투자했느냐'가 아니라 **규모·주도권(운영)·통합성**이다. 단계별로:
- **상류·가스전:** 미쓰비시 = 호주·말레이·브루나이·캐나다 가스전 + 美 셰일(Aethon $5.2B)·Montney 직접 보유 / 한국 = SK E&S 호주 바로사 **37.5%**, 포스코인터 알래스카 추진 — 지분은 있으나 **운영권은 아님**.
- **액화:** 미쓰비시 = 13개 가동 14.9MTPA + Cameron 톨링(설비 없이 액화권) / 한국 = SK E&S 다윈 **25%**, 국내 액화자산 없음.
- **수송:** 미쓰비시 = Coastal GasLink 사용권 + 전용 선단 / 한국 = SK E&S 2019 민간 첫 전용 LNG 선단(국내 수송은 갖췄으나 글로벌 경로통제 약함).
- **도입·터미널:** 한국 = KOGAS 인수기지 + 배관 **5,248km(세계급)** + SK 보령터미널 **350만t** — **인프라는 강함**.
- **판매·트레이딩:** 미쓰비시 = Diamond Gas(싱가포르) 단기매매로 **가격 형성 참여** / 한국 = KOGAS 국내 독점공급 = 가격 '전달' 역할, **글로벌 트레이딩 데스크 부재**.

**한 줄:** 한국은 '인프라(도입·터미널)'는 세계급이나 **'상류 운영권'과 '글로벌 트레이딩'이 비어 가격을 전달만** 하고, 미쓰비시는 전 단계에서 **가격을 형성**한다.

**근거(자체 2차 분석):** 발표자료 단계별 비교표(`reports/lng-compare-winner.pdf`). 기존 개별 노트([[F-SK-ES-수직통합]]·[[F-포스코인터-알래스카LNG]])를 단계축으로 재구성하고 배관 5,248km·보령 350만t·바로사 37.5%·다윈 25% 등 정량 비교를 추가했다. 수치 → `_data/key_metrics.csv`.

> ⚠️ 단계별 지분율·인프라 수치는 자체정리(2026.06)로 일부 1차 재확인 필요. 적대검증 미실시.
