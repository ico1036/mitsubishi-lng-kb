# F · 연결순이익 시계열 (LNG 베타)

**클레임:** 미쓰비시 지배주주귀속 연결순이익은 자원·LNG 가격에 강하게 연동(고베타)된다. **FY2020 1,725억엔(COVID 저점) → FY2021 9,375억엔 → FY2022(Mar2023) 11,806억엔(역대 최고)** 로 단 3년 만에 약 **6.8배** 급등 후, FY2023 9,640 → FY2024 9,507 → FY2025(Mar2026) 8,004억엔으로 정상화. 미쓰비시는 밸류체인 전체(업스트림→액화→운반→트레이딩)를 통합해 이 베타를 *내재화*했다.

**근거(출처 직접 확인):** Flash Report FY2021·FY2022·FY2026 직접 확인(✓), AFR2025로 FY2022~FY2024 교차확인(✓). FY2016~FY2018(4,402→5,602→5,907억엔)은 Flash Report 전년비교치 역산으로 ±5% 오차 가능.
