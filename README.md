## 📂 폴더 안에 뭐가 있나
- `AGENTS.md` — AI 에이전트용 사용 설명서 (사람이 안 읽어도 됨).
- `Mitsubishi-LNG-Vault/` — 지식베이스 본체. `00-INDEX.md`가 목차입니다.
  - `S/` 출처 · `F/` 검증된 발견 · `MOC/` 주제별 묶음 · `_data/` 숫자 데이터(CSV) · `ONTOLOGY-관계지도.md` 관계도.
- `sources-original/` — 빌드 원본(재무 엑셀 + 자체 리포트 HTML 3종, Equity Research 포함). 지식베이스를 만들 때 쓴 원자료.
- `reports/` — 최종 산출물·발표자료(발표/비교 PDF 3종). 사람이 읽거나 발표할 때 보는 결과물 — `reports/README.md`에 안내.
- `.claude/skills/mitsubishi-data/` — 데이터 조회 도구(자동 사용됨).

## ⚠️ 알아둘 점
- 이 자료는 **리서치 정리물이며 투자 자문이 아닙니다.**
- 1차로 확인 안 된 일부 숫자는 자료 안에 "검증 필요"로 표시돼 있습니다.
- 호르무즈 원유 통과 비중처럼 출처마다 값이 다른 항목(35% vs 21%)은 둘 다 적어 뒀습니다.
- 기준 시점: 2026년 6월.

## 🛠 (선택) 직접 열어보거나 도구를 돌리고 싶다면
- **파이썬이 없어도 됩니다.** 모든 데이터는 평문입니다. 그냥 순서대로 열어보세요:
  `Mitsubishi-LNG-Vault/00-INDEX.md`(목차) → `_data/*.csv`(숫자) → `S/ F/ MOC/` 폴더의 `.md`(분석).
- **파이썬(`python3`)이 있다면** 추가 설치 없이 조회 도구를 쓸 수 있습니다:
```bash
python3 .claude/skills/mitsubishi-data/query.py finance

```
